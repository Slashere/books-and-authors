# Каталог книг на Yii2

Тестовое задание: простой каталог книг на Yii2 с авторами, подписками и отчетом о "Топ-10 авторов"

## Функционал

-  **Книги**
   - CRUD операции
   - Загрузка и сохранение обложки в `(uploads/covers/...)`
   - На форме выбора авторов используется `checkboxList`
   - При создании:
     - Создаётся запись книги
     - Обновляются связи с авторами
     - Сохраняется обложка
     - Через `NotificationService` отправляются SMS подписчикам соответствующих авторов

- ️ **Авторы**
   - CRUD операции
   - При удалении автора:
       - Если у автора есть книги, то удаление запрещено
       - Подписки `(subscription)` удаляются каскадно

- ️ **Подписки**
    - Создание по номеру телефона
    - Доступны без авторизации
    - Номер уникален на авторе
    - При повторной подписке на того же автора с существующим номером выдается ошибка

-  **Отчёт**
    - Топ-10 авторов выпустивших больше всего книг за выбранный год
    - Эндпоинт: `/report/top-authors?year=2025`

-  **Права доступа**
   - **Гость** (`?`)
      - Просмотр книг, автора, отчёта Топ-10 авторов
      - Подписка на автора
   - **Юзер** (`@`)
      - Всё то же самое что и гость
      - CRUD книг и авторов
